
# BMI CALCULATOR

A simple and interactive Body Mass Index (BMI) Calculator built with Python. This project allows users to quickly calculate their BMI and understand their health category based on their height and weight. Perfect for beginners to learn Python fundamentals while creating a practical tool!


## Features

-User-friendly input for weight and height
-Calculates BMI using the standard formula
-Provides health category feedback (Underweight, Normal, Overweight, Obese)
-Lightweight, fast, and easy to use

## Installation

Follow these steps to run the BMI Calculator locally using VS Code:

1️⃣ Clone the repository
git clone https://github.com/2006risi-droid/project01.git

This will download the project01 folder to your machine.

2️⃣ Open the project in VS Code
Launch VS Code
Go to File → Open Folder → select project01
Open project1.py in VS Code
3️⃣ Open the terminal in VS Code
Press Ctrl + ` (backtick) or go to View → Terminal
Make sure the terminal is in the project01 folder
4️⃣ Run the BMI Calculator
python project1.py

Or if your system uses Python 3 explicitly:

python project1.py
Enter your weight (kg)
Enter your height (m)
See your BMI and health category directly in the terminal
